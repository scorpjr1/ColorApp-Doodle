按照以下的许可证，我们小组决定调用GitHub网上的免费版简单画图软件。该开发者说明，只要带着开发者的许可证任何人可以随便调用他的作品。因此我们小组按照该软件为根来完成本实习的全部任务，并更该整个软件本来的样式。使在此实习提交的软件代码是大部分我们小组来完成。
